# Linux下配置Git,Node.js+ITK+VTK

## 前言

最近帮研究生写论文,需要ITK做医学领域的开发,因为五花肉喜欢在unix架构下面开发软件,所以此文就来记录我的

### 安装Git

CentOS 7 的yum源中默认已经有git了,如果你是在centos 7下,可以跳过, 安装软件的第一步肯定要已安装依赖包

yum install curl yum install curl-devel yum install zlib-devel yum install openssl-devel yum install perl yum install cpio yum install expat-devel yum install gettext-devel

下载最新的Git包

wget http://www.codemonkey.org.uk/projects/git-snapshots/git/git-latest.tar.gz tar xzvf git-latest.tar.gz cd git-2011-11-30 ＃你的目录可能不是这个 autoconf ./configure make && make install

检查下安装的版本

git —version

安装node.js

由于RHEL的Python版本为2.4， 而Node.js依赖Python27，所以应该先更新一下Python至Python27。

yum install openssl-devel yum install python27 yum install gcc-c++

下载并编译、安装node.js

cd /usr/local/src wget http://nodejs.org/dist/node-latest.tar.gz tar zxvf node-latest.tar.gz cd node-v0.10.18 python2.7 configure make PYTHON=python2.7 make install

安装CMake

curl -L -k https://github.com/Kitware/CMake/archive/master.zip 2> /dev/null > cmake.zip unzip cmake.zip && mv CMake-master CMake cd CMake ./bootstrap make && make install

安装VTK

与安装CMake类似:

curl -L -k https://github.com/Kitware/VTK/archive/master.zip 2> /dev/null > vtk.zip unzip vtk.zip && mv VTK-master VTK mkdir vtk-build cmake ../VTK -DCMake_INSTALL_PREFIX=~/lib/vtk -DBUILD_DOCUMENTATION=OFF -DBUILD_EXAMPLES=OFF -DBUILD_SHARED_LIBS=OFF -DBUILD_TESTING=OFF make && make install

安装ITK

curl -L -k https://github.com/Kitware/ITK/archive/master.zip 2> /dev/null > itk.zip unzip itk.zip && mv ITK-master ITK mkdir itk-build cmake ../ITK -DCMake_INSTALL_PREFIX=~/lib/itk -DBUILD_DOCUMENTATION=OFF -DBUILD_EXAMPLES=OFF -DBUILD_SHARED_LIBS=OFF -DBUILD_TESTING=OFF -DModule_ITKVtkGlue=ON make && make install
